保全api是什么

属于电子数据保全类API，它为用户提供的是电子数据固化保全及出证服务。它是利用专业的电子数据保全技术，将用户上传的电子数据进行实时固化保全，经过保全后的电子数据具有法律效力，为用户产生的法律纠纷提供合法证据。

熊保宝保全API的优势

熊保宝电子数据保全技术由国家电子数据保全技术专利授权开发；

熊保宝电子数据保全技术运用“哈希链滚动加密电子指纹算法”+“区块链技术”+“精确至纳秒级的时间单位”，三重技术保障帮助用户实现电子数据零误差、零丢失、零篡改。

经过熊保宝保全后的电子数据具有权威的法律效力，能够在用户产生法律纠纷时提供合法的呈堂证据
